### 4Ô∏è‚É£ Add the API Key to Your Project

Create a `.env` file in your project root:

```env

```

Your application already loads this using:

```python

load_dotenv()
```

And configures Gemini with:

```python

```

* ‚ùå Never hard-code API keys


## üß™ Common Errors & Fixes

| Error                   | Cause             | Fix                     |
| ----------------------- | ----------------- | ----------------------- |
| `401 Unauthorized`      | Invalid key       | Regenerate API key      |
| `429 Too Many Requests` | Rate limit        | Wait or reduce requests |
| `API key not found`     | `.env` not loaded | Check file location     |
